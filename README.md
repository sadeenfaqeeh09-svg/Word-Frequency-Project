# Word-Frequency-Project
Operating Systems project that finds the top 10 most frequent words using naive, multiprocessing, and multithreading approaches.

## 🖥️ Environment

- OS: Linux (Virtual Machine)
- Virtual Machine: Oracle VM VirtualBox
- CPU: 4 Cores
- RAM: 16 GB
- Programming Language: C
- IDE: CodeBlocks

> Note: The project was implemented and tested on a Linux virtual machine to support multiprocessing and threading features properly.
